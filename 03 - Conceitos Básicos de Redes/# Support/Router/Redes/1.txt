+ SSH +
+++++++

+ Obs: Configurar IP público anteriormente para o roteador (ver rotas)

Router>enable 
Router#configure terminal 
Router(config)#username Diego password cisco1
Router(config)#enable secret cisco2
Router(config)#hostname CENTRAL
CENTRAL(config)#ip domain-name CYBERSECURITY.NET
CENTRAL(config)#ip ssh version 2
CENTRAL(config)#crypto key generate rsa
CENTRAL(config)#line vty 0 15
CENTRAL(config-line)#transport input ssh
CENTRAL(config-line)#login local
CENTRAL(config-line)#end
CENTRAL#wr

+ Monitoramento do tráfego

CENTRAL(config)#exit
CENTRAL(config)#interface fastEthernet 5/0

CENTRAL(config)#exit
